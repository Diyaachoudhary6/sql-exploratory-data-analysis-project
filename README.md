# SQL Exploratory Data Analysis (EDA) Project

## 📌 Project Overview

This project demonstrates **Exploratory Data Analysis (EDA)** using SQL on a Data Warehouse dataset. The objective is to explore the database structure, understand business dimensions, analyze sales performance, evaluate key business metrics, and identify top-performing products and customers using SQL queries. 
Performed exploratory data analysis (EDA) using SQL to uncover trends, identify data quality issues, and generate business insights through filtering, aggregations, joins, window functions, and CTEs.
A comprehensive collection of SQL scripts for data exploration, analytics, and reporting. These scripts cover various analyses such as database exploration, measures and metrics, time-based trends, cumulative analytics, segmentation, and more.

---

# 🎯 Project Objectives

* Explore the overall database structure.
* Understand customer, product, and sales dimensions.
* Analyze historical sales data.
* Calculate important business KPIs.
* Identify customer and product performance.
* Perform ranking and distribution analysis.
* Practice SQL queries used in real business environments.
